ABOUT

会社概要

CODE STARとは

「プログラミングを学ぶ人に最高のエンジニア能力を授ける」というミッションを掲げ、ソフトウェアエンジにリアリングの経験が豊富な少数精鋭のメンターによって高品質なプログラミング教育を実現しました。

FEATURE

特徴

CODESTARは、プログラミング学習者がスキル向上を目指すべく創設されたコミュニティです。
多くの優秀な現役エンジニアがメンターとして在籍していますので、日々の学習の中で生じた疑問/質問を気軽に相談していただけます。
また、週に１度の勉強会や、メンバー同士の交流を促す企画なども多数ご用意しております。

勉強会

月1の勉強会などを通じて 技術と知識を研鑽し続けます。

交流会

メンバー同士の交流会には現役のエンジニアから初心者の方まで幅広い層の方が集まります。

SERVICE

事業内容

プログラミング学習教材【CODESTAR】では、優れた講師による本質的なエンジニアリング能力を実現するためのプログラミング学習カリキュラムを提供します。
CODESTARではIT業界の第一線で活躍しているエンジニアを揃えております。
ITベンチャーの現役エンジニア、元プログラミングスクールのメンターやフリーランスエンジニアなど、学習者のさまざまな質問/疑問に回答できるよう最高の環境を皆さんに提供します。
